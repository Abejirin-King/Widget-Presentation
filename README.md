# SingleChildScrollView Demo

## Description
SingleChildScrollView is a widget used to avoid overflow in a Flutter application by having a scrollable widget for user input 

## Run Instructions

1. Clone the repo
2. Run flutter pub get
3. Run flutter run

## Properties 

### scrollDirection
 Changes the scroll direction from vertical to horizontal.

### reverse
Allows the scroll direction to be reversed.

### padding
Provides additional spacing around the content of the scrollable item(s).
